A small note on how we did it

This data was collected from Indiegogo starting at 20:25 E.S.T. on 20 May until the end of the campaign. The campaign was at $81,408 at that point. Therefore, these statistics should be considered a biased sample.

We used a PHP script on a cron job with PHP Simple HTML DOM Parser to collect this data. The Google Charts API was used to create most of these infographics.
